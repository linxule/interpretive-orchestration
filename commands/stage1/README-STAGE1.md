## How to Complete Stage 1

### 1. Sample Your Data
- 10-15 documents (purposeful diversity)
- Use `/qual-think-through` to plan sampling strategy

### 2. Code Manually
- Stay close to participant language (informant-centric)
- Build 1st-order concepts from their terms
- Don't worry about perfection - patterns will emerge

### 3. Write Memos
- Document emerging insights
- Question your assumptions
- Connect to theory
- Save in `stage1-foundation/memos/`

### 4. Build Initial Framework
- Group 1st-order codes into 2nd-order themes
- Develop preliminary aggregate dimensions
- Use `skills/gioia-methodology/templates/gioia-data-structure-template.json`
- Save to `stage1-foundation/initial-structure.json`

### 5. Check Readiness
When you feel:
> "I have a sense of my data. I know what patterns matter to ME.
> I can distinguish superficial from theoretically significant.
> I have categories that emerged from my engagement."

**Then:** Run `/qual-status` - it will validate Stage 1 completion
